# Bookworm Visual Basic 6
Wersja programu do zarządzania biblioteką, Bookworm, stworzony w języku Visual Basic 6.0 <br><b>Wspiera od Windows 95 do Windows 11 (Windows 95, Windows 98, Windows NT 4.0 and 4.1+, Windows 2000, Windows ME, Windows XP, Windows Vista, Windows 7, Windows 8 and 8.1, Windows 10, Windows 11)</b>
<br><b>Wspiera Windows Server: Windows Server NT 4.0, Widows Server 2000, Widows Server 2003, Widows Server 2008, Widows Small Business Server 2008, Widows Server 2012, Widows Server 2012 R2, Widows Server 2016, Widows Server 2019, Widows Server 2022, Widows Server 2025
<br><i>Note: Windows 95 and Windows 98 requires MSVCVB6.dll runtime DLL to function, which is selected in the installer by default</i>
<br><i>Uwaga: Windows 95 i Windows 98 wymaga biblioteki MSVCVB6.dll aby działać, która jest automatycznie zaznaczona w instalatorze</i>

# Założenia
* Prędkość i efektywność użycia
* Prywatność - niemożność podejrzenia danych przez osoby nieautoryzowane
* Łatwia instalacja i przenośność - brak zewnętrzynych wymagań od użytkownika
* Łatwość użytku dla użytkowników, lecz wielkie możliwości zarządzania dla superadminów
* Przeszłe, teraźniejsze i przyszłe wsparcie Windows

# Dlaczego?
* Orginalna wersja w języku Python zawierała wiele zaśmiecenia w kodzie, kompilacja utrudniona problematycznymi bibliotekami
* Visual Basic ma większe wsparcie na Windows - działa od Windows 98 po Windows 11 (wspiera Windows 98, Windows NT 4.0 and 4.1, Windows 2000, Windows ME, Windows XP, Windows Vista, Windows 7, Windows 8 and 8.1, Windows 10, Windows 11) oraz Windows Server NT 4.0, Widows Server 2000, Widows Server 2003, Widows Server 2008, Widows Small Business Server 2008, Widows Server 2012, Widows Server 2012 R2, Widows Server 2016, Widows Server 2019, Widows Server 2022, Widows Server 2025
* Podczas użytkowania wersji Python zauważyłem bardzo wiele poprawek QoL
